Bitácora del Proyecto

Registro de Actividades

Evolución de las tareas y fases implementadas en el desarrollo de este trabajo prácrico.

Fase 1: Estructura del Proyecto

Se organizó el directorio principal creando la estructura base y se añadieron los archivos index.html y los de lps pergiles individuales en la raíz. Asimismo, se agregaron en la raiz la carpeta css con la hoja de estilos styles.css, la carpeta img para los recursos gráficos y la carpeta js. Esta última contiene los archivos desplegable.js el que contiene la lógica para alternar la visibilidad de los detalles en las tarjetas y el archivo menu-hamburguesa.js que controla la navegación

Fase 3: Diseño y Tipografía

Se enlazaron correctamente las tipografías Montserrat y Roboto a través de Google Fonts y se verificó su aplicación en la interfaz. El layout se organizó con Flexbox para adaptarse a dispositivos.

Fase 4: Estructuración HTML

Maquetación de las secciones principales. Se agregaron textos alternativos (alt) en todas las imágenes para cumplir con la accesibilidad.

Fase 2: Repositorio y Publicación

Se creó y configuró el repositorio en GitHub, subiendo los archivos del proyecto. Se realizó la publicación en línea a través de Vercel.

Proyecto realizado para la materia Desarrollo de Sistemas Web (Front End) - IFTS N.° 29
